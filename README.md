# Панибратский Дмитрий - Профессиональное портфолио

## Структура проекта

- `index.html` - Главная страница
- `стиль.css` - Основные стили
- `expertise.html` - Секция экспертиз (AI и Цифровая трансформация)
- `expertise.css` - Стили для секции экспертиз
- `calculators.html` - Страница с калькуляторами
- `certificates.html` - Страница с сертификатами
- `tools.html` - Инструменты
- `wiki.html` - База знаний
- `wiki.json` - Данные для вики
- `js/certificates.js` - JavaScript для сертификатов
- `images/` - Папка с изображениями

## Направления экспертизы

### AI & Предиктивная аналитика
- AI-прогнозирование спроса
- Предиктивная аналитика рисков

### Цифровая трансформация SCM
- Цифровые двойники процессов
- Автоматизация E2E-процессов

## Технологии
- HTML5
- CSS3 (Grid, Flexbox)
- JavaScript
- SVG иконки
- Адаптивный дизайн

## Установка
1. Клонировать репозиторий
2. Открыть `index.html` в браузере
3. Для разработки - использовать локальный сервер
